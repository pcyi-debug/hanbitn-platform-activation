# 01. 문제 정의 (Problem Definition)

## 1. 한 줄 정의

> 한빛앤(주) 플랫폼의 **활성화 지표가 일정 수준에서 더 이상 성장하지 못하고 정체**되어 있으며, 신규 유입이 실제 활성 사용자로 전환되지 못하고 이탈하고 있다.

## 2. 정체의 증상 (Symptoms)

정체는 보통 단일 지표가 아니라 여러 지표가 동시에 멈추는 형태로 나타납니다. 아래는 진단 시 확인해야 할 증상 체크리스트입니다.

- [ ] 신규 가입 수는 유지되나 **DAU/MAU 비율(스티키니스)** 이 상승하지 않음
- [ ] 가입 후 **첫 핵심 행동(Aha Moment) 도달률** 이 낮거나 하락
- [ ] **D1 / D7 / D30 리텐션 곡선** 이 바닥에서 평탄화
- [ ] 마케팅 비용 투입 대비 활성 사용자 증가폭 둔화 (CAC 상승)
- [ ] 핵심 기능 사용 빈도가 특정 사용자군에 편중

## 3. 문제의 범위 (Scope)

| 포함 (In Scope) | 제외 (Out of Scope) |
|------------------|----------------------|
| 가입 → 첫 핵심행동 → 재방문 구간의 활성화 | 신규 시장 진출/해외 확장 |
| 온보딩 경험 및 초기 사용자 여정 | 가격 정책 전면 개편 |
| 리텐션 및 재참여(Re-engagement) 구조 | 신규 BM(수익모델) 발굴 |
| 활성화 관련 데이터 측정 체계 | 조직 개편 |

## 4. 영향 (Business Impact)

활성화 정체가 지속될 경우 예상되는 영향:

1. **성장 한계**: 유입을 늘려도 매출/사용량이 비례 성장하지 않음
2. **마케팅 효율 악화**: 새는 독에 물 붓기 → CAC 회수 기간 증가
3. **제품 신뢰 저하**: 낮은 리텐션이 입소문·추천을 약화시켜 자연 유입 감소
4. **의사결정 지연**: 정체 원인 미규명 시 잘못된 곳에 자원 투입

## 5. 핵심 질문 (Key Questions)

문제 해결을 위해 반드시 답해야 할 질문들:

1. 사용자가 가입 후 **"가치를 처음 느끼는 순간(Aha Moment)"** 은 무엇이며, 몇 %가 도달하는가?
2. 정체가 **유입 품질** 문제인가, **제품 경험** 문제인가, **재참여** 문제인가?
3. 어떤 사용자 세그먼트가 활성화되고, 어떤 세그먼트가 이탈하는가?
4. 활성화 퍼널의 **가장 큰 누수 지점(Drop-off)** 은 어디인가?

## 6. 성공의 정의 (Definition of Success)

본 프로젝트가 성공했다고 판단하는 기준:

- 활성화 퍼널 핵심 단계 전환율 측정 체계 구축 완료
- 최소 1개 이상의 퍼널 누수 구간에서 유의미한 개선 검증
- DAU/MAU 스티키니스 및 D7 리텐션의 추세 반전(정체 → 상승)

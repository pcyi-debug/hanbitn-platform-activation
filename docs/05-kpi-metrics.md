# 05. KPI / 지표 체계 (Metrics Framework)

활성화 정체 해결의 성패는 **측정 가능성**에 달려 있습니다. 본 문서는 북극성 지표부터 실험 지표까지의 체계를 정의합니다.

## 1. 지표 위계 (Metric Hierarchy)

```
North Star Metric (북극성 지표)
        │
   ┌────┴────┐
 입력지표   결과지표
(Leading)  (Lagging)
```

### 북극성 지표 (North Star Metric)
> **주간 활성 가치경험 사용자 수 (Weekly Activated Users)**
> = 한 주 동안 핵심 가치 행동(Aha)을 1회 이상 수행한 순수 사용자 수

핵심 가치와 직결되며, 활성화·리텐션·성장을 동시에 반영합니다.

## 2. 활성화 핵심 지표

| 지표 | 정의 | 목표 방향 |
|------|------|:---------:|
| 온보딩 완료율 | 가입자 중 온보딩 완료 비율 | ▲ |
| **Aha 도달률** | 가입자 중 첫 핵심행동 도달 비율 | ▲ |
| Time-to-Value | 가입 → 첫 핵심행동까지 소요 시간 | ▼ |
| 첫날 활성화율 | 가입 당일 핵심행동 수행 비율 | ▲ |

## 3. 리텐션 지표

| 지표 | 정의 | 비고 |
|------|------|------|
| D1 / D7 / D30 리텐션 | N일 후 재방문 비율 | 코호트별 추적 |
| DAU / MAU (스티키니스) | 일간/월간 활성 비율 | 습관화 강도 |
| 리텐션 곡선 평탄화 지점 | 곡선이 안정되는 시점·수준 | 핵심 사용자 비율 |

## 4. 퍼널 전환 지표

| 단계 | 전환 지표 |
|------|-----------|
| 가입 → 온보딩 완료 | 온보딩 전환율 |
| 온보딩 → 첫 핵심행동 | 활성화 전환율 |
| 첫 행동 → 재방문(D1) | 초기 리텐션 전환율 |
| 재방문 → 습관화(D7+) | 습관 전환율 |

## 5. 선행 지표 vs 후행 지표

| 선행 지표 (Leading) | 후행 지표 (Lagging) |
|----------------------|----------------------|
| 온보딩 완료율 | D7 리텐션 |
| 알림 클릭률 | 주간 활성 사용자 수 |
| 첫 행동 도달률 | DAU/MAU |
| 추천 발송 수 | 추천 경유 가입 수 |

> 선행 지표는 **빠르게 움직여 실험 방향을 알려주고**, 후행 지표는 **최종 성과를 확인**합니다.

## 6. 대시보드 설계 (Dashboard Spec)

| 대시보드 | 구성 요소 | 갱신 주기 |
|----------|-----------|:---------:|
| 활성화 퍼널 | 단계별 전환율·누수율 | 일간 |
| 코호트 리텐션 | 가입 주차별 리텐션 히트맵 | 주간 |
| 세그먼트 | 활성 vs 이탈 사용자 프로파일 | 주간 |
| 실험 추적 | 진행 중 A/B 실험 결과 | 실험별 |

## 7. 지표 운영 원칙

1. **하나의 북극성** — 조직 전체가 같은 방향을 본다
2. **허무 지표(Vanity Metric) 경계** — 누적 가입 등 행동을 바꾸지 않는 지표 지양
3. **세그먼트로 쪼개 본다** — 평균은 진실을 숨긴다
4. **목표(Target) 명시** — 모든 핵심 지표에 기간별 목표 수치 설정

## 8. 측정 목표 예시 (Target)

| 지표 | 현재(가정) | Q1 목표 | Q2 목표 |
|------|:----------:|:-------:|:-------:|
| Aha 도달률 | 30% | 45% | 55% |
| D7 리텐션 | 12% | 17% | 22% |
| DAU/MAU | 18% | 22% | 26% |

> 현재 수치는 실제 데이터 진단 후 확정합니다.
